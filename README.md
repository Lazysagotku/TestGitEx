1. Текстовый файл, содержащий все варианты запросов, пронумерованно. Вложить его в проект.
2. Логирование запросов к БД. Пусть пишется в текстовый файл в рабочем каталоге, в подкаталоге "Logs". Примерно в таком формате:
<дата и время запроса> <БД> <идентификатор запроса> <время выполнения запроса в секундах>
Где:
<БД> - система IS / J
<идентификатор запроса> - номер запроса из текстового файла (п.1)
3. В настройках - кнопку, открывающую подкаталог с логами.

Это позволит понимать, как можно будет оптимизировать работу приложения.
